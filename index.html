The race

  • See Track map on holmenkollstafetten.no
  • We think you will be able to use the same app as last year for tracking your team's exchanges.
  • Be sure to meet the persons you are exchanging with before the race
  • Look for the green shirt
  • Remember: The one that is receiving the stick, is responsible for the exchange
  • If you dont find anyone at the end of your run, shout Cisco loud. If all else fails, you may need to run the next stage too, but this doesn't happen often.
  • There are usually places you can leave your luggage at each exchange, but it's lightly guarded, so don't bring too valuable stuff.
